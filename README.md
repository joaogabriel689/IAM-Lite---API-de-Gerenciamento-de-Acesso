# IAM Lite - API de Gerenciamento de Acesso

API REST desenvolvida com Django Rest Framework para gerenciar
usuários, solicitações de acesso e auditoria de ações sensíveis.

## Funcionalidades
- CRUD de usuários
- Solicitação de acesso
- Aprovação/Rejeição
- Logs de auditoria
- Controle de permissões (em andamento)

## Tecnologias
- Python
- Django
- Django Rest Framework
- SQLite (dev)
